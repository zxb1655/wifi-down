# WiFi 跑流量工具

基于 Electron 的 Windows 桌面工具，自动扫描并连接指定 WiFi 设备，跑流量后上报测速结果。

## 使用方式

```bash
npm install
npm start
```

## 操作步骤

1. 在顶部填写 API 地址（baseURL）
2. 可选：配置备用 WiFi 名称和密码（用于上报时网络不通的情况）
3. 点击"获取设备"从接口拉取待测速设备列表
4. 点击"扫描 WiFi"查看附近的 WiFi 网络
5. 点击"开始跑量"执行完整流程：连接 WiFi → 跑流量 → 上报结果
6. 可选：配置定时器自动定期执行跑量任务

## 功能说明

- 自动从 API 获取待测速 WiFi 设备列表
- 扫描电脑附近 WiFi 并自动匹配设备
- 每个设备随机跑 100~200MB 流量
- 跑量完成后通过 API 上报结果
- 支持备用 WiFi（上报时无网络自动切换）
- 支持定时自动执行
- 实时进度展示和运行日志

## 注意事项

- 需要以管理员权限运行才能操作 WiFi（netsh 需要权限）
- 仅支持 Windows 系统
